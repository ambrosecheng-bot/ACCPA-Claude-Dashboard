Odoo Projects integration

Connect Claude to Odoo Projects via MCP. Live WIP, deadlines and task status feed automatically into the management view — no manual updates.

Odoo connection — 4 steps
  1. Confirm Odoo hosting type (Online / .sh / on-prem) and enable API key access
  2. Choose connector route: hosted MCP (e.g. Pantalytics — free beta) or open-source self-host (free)
  3. Add connector in Claude Settings and authorise via OAuth
  4. Start read-only on Odoo Projects; widen to Contacts & Documents after review
